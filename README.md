# n8n Quickstart
> Setup n8n locally in a Docker containers

Follow this guide to configure n8n to access Ollama:

- [Ollama docs - n8n](https://docs.ollama.com/integrations/n8n)

## Setup

- Install Docker.
- Install Ollama and start it on the host.

## Start service

```sh
$ docker compose up
```

Then open at:

- http://localhost:5678/
